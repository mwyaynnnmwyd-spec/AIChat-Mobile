# AIChat-Mobile 🤖💬

تطبيق دردشة ذكي مع خدمة ذكاء اصطناعي مجانية من Hugging Face

## 🎯 المميزات

✅ دردشة ذكية مع AI حقيقي
✅ واجهة احترافية وبسيطة
✅ يدعم العربية والإنجليزية
✅ مجاني 100% بدون رسوم
✅ بدون الحاجة إلى مفتاح API
✅ يعمل بدون إعلانات
✅ اتصال سريع وآمن
✅ سهل الاستخدام

---

## 📱 المتطلبات

- Android 5.0 أو أعلى
- اتصال إنترنت
- Sketchware أو Android Studio

---

## 🚀 طريقة التثبيت

### الطريقة الأولى: باستخدام Sketchware

1. افتح Sketchware
2. انقر على "Create New Project"
3. اختر تصميم فارغ
4. انسخ ملف `MainActivity.java` إلى Custom Code
5. انسخ ملف `activity_main.xml` إلى Layout
6. أضف الـ permissions من Manifest
7. اضغط RUN

### الطريقة الثانية: باستخدام Android Studio

1. استنسخ المستودع
2. افتح المشروع في Android Studio
3. اضغط Run

## 🌐 API المستخدم

المشروع يستخدم **Hugging Face Inference API** وهي:
- مجانية 100%
- لا تحتاج مفتاح تسجيل
- سريعة وموثوقة
- تدعم نماذج ذكاء اصطناعي متعددة
